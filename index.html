Information

During my PhD I experimented with the idea of writing my thesis in a public code repository. I hoped that by making my work on my thesis public, my time would become more accountable by the fact that anyone would be able to see when I was or was not working. This didn't really work out as planned, but the PhD was completed eventually.

I also believe that all academic research should be made more open. This is a small step towards such a philosophy. (Making my code itself available would be better again, but one step at a time. The bulk of my magnetic force code is indeed available here.)

In the Github repository for this work I've started to list open issues that I'm aware of, such as typos and (gasp) mistakes.
